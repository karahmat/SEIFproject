Instructions

  1. Ensure that all the players are on the page before the first player clicks on Create New Game
  2. Once the first player clicks on Create New Game, a five-characters Game Room Code will appear
  3. The first player then shares the Game Room Code to the other players
  4. The other players paste the Game Room Code in the field "Enter Game Code"
  5. The last player to join the game will see a green "Start Game Now" button. Pressing the button will trigger the WOF game
